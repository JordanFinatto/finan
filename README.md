# finan
Tema da atividade prática de GERÊNCIA DE CONFIGURAÇÃO DE SOFTWARE: controle de despesas pessoais (receitas e despesas, entradas e saídas)

Aplicação construída com base em:
- PHP 7.3.7
- Laravel 4.2.10
- Bootstrap 4
- PHPUnit 9.5
- Composer 2.3.5
- XAMPP 7.3.7
